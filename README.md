    <div class="command-section">
        <h2>1. üÜï git init</h2>
        <p><strong>Use:</strong> Initializes a new Git repository.</p>
        <p><strong>Example:</strong></p>
        <pre><code>git init</code></pre>
        <p>This creates a <code>.git</code> directory in your project, making it a Git repository.</p>
    </div>

    <div class="command-section">
        <h2>2. üìÇ git clone</h2>
        <p><strong>Use:</strong> Clones an existing repository to your local machine.</p>
        <p><strong>Example:</strong></p>
        <pre><code>git clone https://github.com/user/repo.git</code></pre>
        <p>This command creates a copy of the repository on your local machine.</p>
    </div>

    <div class="command-section">
        <h2>3. ‚ûï git add</h2>
        <p><strong>Use:</strong> Stages changes for the next commit.</p>
        <p><strong>Example:</strong></p>
        <pre><code>git add index.html</code></pre>
        <p>This stages <code>index.html</code> for the next commit.</p>
    </div>

    <div class="command-section">
        <h2>4. üìå git commit</h2>
        <p><strong>Use:</strong> Records changes to the repository.</p>
        <p><strong>Example:</strong></p>
        <pre><code>git commit -m "Add index.html"</code></pre>
        <p>This creates a commit with the message "Add index.html".</p>
    </div>
